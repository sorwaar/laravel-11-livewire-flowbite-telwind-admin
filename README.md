- [NPM](https://www.npmjs.com/)

## Installation

1. Clone the repository `git clone https://github.com/devprantoroy/locker-web.git`
2. Run `composer install`
3. Run `npm install`
4. Run `npm run dev`
5. Run `php artisan serve`
6. Visit `http://localhost:8000` in your browser
